# Ts Rest Server

A TypeScript Rest Server with Node.js

## Description

A TypeScript Rest Server with Node.js

## Getting Started

### Dependencies

- `npm`
- `node`

### Installing

- Clone the repository and install dependencies:

```
git clone https://github.com/Felipe-Navas/ts-rest-server.git && cd ts-rest-server && npm install
```

### Executing program

```
npm start
```

### Running tests

```
npm test
```
